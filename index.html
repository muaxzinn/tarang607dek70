ตั้งค่าช่องทางการแจ้งเตือน

แจ้งเตือนตารางเรียนวันพรุ่งนี้ล่วงหน้า (19:00 น.)

ระบบจะรวบรวมสรุปรายชื่อวิชา เวลาเรียน และผู้สอนของวันถัดไป ส่งผ่าน LINE Notify (และ Discord Webhook ถ้าตั้งค่า) ทุกวันเวลา 19:00 น. (1 ทุ่ม) โดยอัตโนมัติเมื่อเปิดหน้าจอนี้ทิ้งไว้

LINE Notify (การส่งเตือนกลุ่มไลน์)

1. ไปที่เว็บไซต์ LINE Notify ล็อกอินให้เรียบร้อย

2. กดออกรหัส Token เลือกกลุ่มนักเรียนห้อง 6/7 นำรหัสมาวางด้านล่าง

Discord Webhook URL (ส่งเตือนเข้ากลุ่มดิสคอร์ด)

คัดลอกลิงก์ Webhook จากแชทห้องกลุ่มห้องเรียนของคุณมาใส่ตรงนี้ เพื่อให้ข้อความ Embed สวยงามถูกส่งเข้าไปให้เพื่อนๆ เมื่อถึงเวลาเรียน

เสียงขานคาบเรียน (Thai TTS)
แจ้งเตือนผ่านบราวเซอร์

ปรับเปลี่ยนข้อมูลวิชาเรียนในตาราง

เลือกวันเรียน คาบเรียน และพิมพ์ข้อมูลวิชาใหม่ที่คุณต้องการปรับแต่งเพื่อทดแทนข้อมูลวิชาในวันดังกล่าว

ล้างการตั้งค่าและคืนค่าระบบ

ปุ่มนี้จะล้างข้อมูลการปรับแต่งเวลาเรียน ข้อมูลรายวิชา ข้อมูลแจ้งเตือน บันทึกการบ้าน และโทเคนทั้งหมดที่เก็บไว้ในเบราว์เซอร์ของคุณ และจะโหลดตารางเรียน 6/7 ดั้งเดิมตามรูปภาพ **"ตารางเรียน.jpg"** กลับขึ้นมาทำงาน

คำเตือนที่สำคัญ

การคืนค่านี้จะมีผลทันทีและไม่สามารถกู้ข้อมูลเดิมคืนได้อีกหลังจากดำเนินการเสร็จสิ้น
